# Multi-Target Model Training

## Objective

Train separate machine learning models for:

- Underweight Prediction
- Stunting Prediction
- Wasting Prediction

## Model Used

Random Forest Classifier

## Features

- Age
- Sex
- Weight
- Height
- MUAC
- WAZ
- HAZ
- WHZ
- BMI

## Output

Three trained models were created and saved for deployment in the pediatric malnutrition detection system.
