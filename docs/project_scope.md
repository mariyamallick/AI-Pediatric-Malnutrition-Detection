# Project Scope

## Project Title

AI-Assisted Pediatric Malnutrition Detection and Personalized Nutritional Recommendation System

## Target Population

Children aged 0–59 months (under five years).

## Objective

Develop an AI-powered clinical decision support system that predicts pediatric malnutrition using anthropometric measurements and provides personalized nutritional recommendations based on WHO guidelines.

## Expected Output

- Nutritional Status Prediction
- Malnutrition Classification
- Growth Assessment
- Personalized Nutrition Advice
- PDF Report for Healthcare Professionals

## Limitations

The system is designed to assist healthcare professionals and caregivers. It does not replace professional medical diagnosis or treatment.
